# Automated-Theorem-Prover

# Description
This project demonstrates a logical theorem parser and vailidity checker using Python. Additionally, it constructs a binary tree and truth table for each logical expression.

This project was done for CSC 210 - Discrete Mathematics (in a group of 2)
